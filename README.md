# Fashion-MNIST GAN

## Project Overview

This project implements a simple Generative Adversarial Network (GAN) using TensorFlow and Keras to generate Fashion-MNIST clothing images.

The GAN consists of two neural networks:

* Generator
* Discriminator

The Generator learns to create fake clothing images from random noise vectors.

The Discriminator learns to distinguish between real Fashion-MNIST images and fake images generated by the Generator.

Both networks compete against each other during training, gradually improving the quality of generated images.

---

## Dataset

Fashion-MNIST

* 60,000 training images
* 10 clothing categories
* Image size: 28 × 28 pixels
* Grayscale images

---

## Generator Architecture

Input:

* Noise Vector (100 dimensions)

Architecture:

* Dense(256)
* LeakyReLU
* Dense(512)
* LeakyReLU
* Dense(1024)
* LeakyReLU
* Dense(784, activation="tanh")
* Reshape(28,28,1)

Output:

* Generated Fashion-MNIST image

---

## Discriminator Architecture

Input:

* Image (28×28×1)

Architecture:

* Flatten
* Dense(512)
* LeakyReLU
* Dense(256)
* LeakyReLU
* Dense(1, activation="sigmoid")

Output:

* Probability of being a real image

---

## Training Strategy

Discriminator:

* Real Images → Label 1
* Fake Images → Label 0

Generator:

* Fake Images → Label 1

The Generator attempts to fool the Discriminator while the Discriminator attempts to correctly classify real and fake images.

## Results

The Generator successfully learned basic clothing structures such as:

* Shirts
* Coats
* Trousers

Generated images remained relatively blurry due to the use of a fully connected GAN architecture.

Future improvements include implementing a Deep Convolutional GAN (DCGAN).
